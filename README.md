# AI Marketing Revenue Prediction System

## Overview
This project predicts marketing campaign revenue using Machine Learning algorithms and an interactive AI dashboard.

## Features
- Revenue prediction using ML
- Interactive analytics dashboard
- Campaign performance analysis
- Multiple ML model comparison
- Data visualization
- Real-time prediction interface

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Plotly

## Machine Learning Models
- Linear Regression
- Random Forest
- Decision Tree
- KNN Regressor

## Project Structure
- train_model.py → ML training pipeline
- app.py → Interactive dashboard
- data/ → Dataset files
- model/ → Saved ML models
